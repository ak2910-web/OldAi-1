# VedAI Backend

## Local Development

- Start emulators: `npm run serve`
- Deploy to Firebase: `npm run deploy`
- Production deploy: `npm run deploy:prod` (edit project id)

## Notes
- Requires Node.js 18+
- Add your API keys to `.env`
- Do not commit `.env` or `node_modules`
- For production, use the Blaze plan for external API calls
